# Compilation and Execution Instructions

## For p3:

1. Compile the code:
   ```bash
   cc p1.c
   ```
   or for p2:
   ```bash
   cc p3.c
   ```

2. Execute the compiled program:
   ```bash
   ./a.out
   ```

## For p2 and p1:

1. Compile the code:
   ```bash
   cc p2.c
   ```

2. Execute the compiled program:
   ```bash
   ./a.out "filename"
   ```

## For p4:

1. Compile the code:
   ```bash
   cc p4.c
   ```

2. Execute the compiled program:
   ```bash
   ./a.out "filename"
   ```

## For p5:

1. Compile the code:
   ```bash
   cc p5.c
   ```

2. Execute the compiled program with the required file as input:
   ```bash
   ./a.out file.txt
   ```

## For p6:

1. Compile the code:
   ```bash
   cc p6.c
   ```

2. Execute the compiled program with the required file and character number:
   ```bash
   ./a.out file.txt char_num
   ```
